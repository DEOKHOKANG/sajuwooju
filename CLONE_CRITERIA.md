# 복제 등급 기준 (Clone-Grade Criteria)

## 정의
복제 등급이란 원본 사이트와 **95% 이상의 시각적/기능적 일치도**를 달성한 상태를 의미합니다.

## 평가 기준

### 1. 시각적 정확도 (Visual Accuracy) - 40%
- [ ] 색상 정확도 ≥99% (RGB 값 ±2 이내)
- [ ] 폰트 크기, 두께, 행간 정확도 100%
- [ ] 간격(margin, padding) 정확도 ±2px 이내
- [ ] Border radius, shadow 정확도 100%
- [ ] 레이아웃 구조 100% 일치

### 2. 반응형 동작 (Responsive Behavior) - 20%
- [ ] 모바일 (375px) 레이아웃 100% 일치
- [ ] 태블릿 (768px) 레이아웃 100% 일치
- [ ] 데스크탑 (1920px) 레이아웃 100% 일치
- [ ] 브레이크포인트 전환 애니메이션 일치

### 3. 인터랙션 (Interaction) - 20%
- [ ] 모든 버튼 hover/active 상태 일치
- [ ] 애니메이션 타이밍, 이징 100% 일치
- [ ] 스크롤 동작 일치
- [ ] 모달/팝업 동작 일치

### 4. 컴포넌트 충실도 (Component Fidelity) - 15%
- [ ] 모든 컴포넌트 완전 구현
- [ ] 컴포넌트 구조 원본과 동일
- [ ] Props/상태 관리 정확

### 5. 디테일 (Details) - 5%
- [ ] 아이콘 정확도
- [ ] 이미지 크기/비율
- [ ] 텍스트 정렬
- [ ] 미세한 시각적 요소

## 불합격 사례
- "비슷하게" 구현 ❌
- "대략적으로" 일치 ❌
- "거의 같음" ❌
- 임의의 색상/간격 사용 ❌
- 스크린샷 없이 구현 ❌

## 합격 기준
- Pixel-perfect 매칭 ✅
- 스크린샷 대조 검증 통과 ✅
- 모든 viewport에서 검증 ✅
- 측정 가능한 정확도 ≥95% ✅
